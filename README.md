 # Building a Differential Drive Robot from Scratch


## Overview

The main goal of this project is to build a basic robotics platform from scratch. This will allow me to learn the fundamentals of **robotics, mechanics, electronics**, and **microcontroller programming**.


> My long term goal is to build a fully autonomous robot with SLAM (Simultaneous Localization and Mapping) on my own.


---


## Key Objectives

* **C++ Mastery:** Implementing control logic in pure C++ to understand low-level hardware interaction.

* **Mobile Control:** Developing an interface to control the robot via a smartphone application.

* **Knowledge Expansion:** Bridging the gap between theory and practice in mechanical design and circuit building.
